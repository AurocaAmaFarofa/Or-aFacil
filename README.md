# OrçaFácil

A verdade é que calcular preço de projeto na cabeça ou ficar brigando com planilha no celular sempre dá dor de cabeça. O **OrçaFácil** nasceu para resolver exatamente isso de um jeito simples, rápido e direto ao ponto.
Ele é um sistema web pensado para quem trabalha por conta própria — seja você marceneiro, confeiteiro, designer ou mecânico. A ideia é automatizar aquela matemática de padaria (insumos + horas de trabalho + margem de lucro) para você saber exatamente quanto cobrar em segundos.
